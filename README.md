# Dotfiles
Configuration for niri + noctalia shell + kitty + nvim.

- OS: Arch Linux
- WM: niri 26.04 (Wayland)
- Theme: Adwaita [GTK2], adw-gtk3-dark [GTK3/4]
- Icons: Papirus-Dark [GTK2/3/4]
- Font: Adwaita Sans (11pt) [GTK2], Noto Sans (11pt) [GTK3/4]
- Cursor: Adwaita (24px)
- Terminal: kitty 0.46.2
- Terminal Font: JetBrainsMonoNF-Regular (11pt)
- CPU: AMD Ryzen 5 3600 (12) @ 4.21 GHz
- GPU: NVIDIA GeForce GTX 1660 SUPER
